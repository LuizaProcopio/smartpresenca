<h1>
  SmartPresença - Automação da Chamada nas Salas de Aula
</h1>

Repositório desenvolvido para fins de documentação de processos.

## Objetivo
Desenvolver um Sistema de automação da chamadas nas salas de aula, onde seja possível:

1) Ter dois tipos de login

   1.1) Professor
   
   1.2) Aluno

2) Geração do QR Code

   2.1) Professor irá liberar um código, o qual aluno irá inserir e então terá um QR Code válido por 10 minutos.

   2.2) Presença será validada ao fazer a leitura do QR Code na ESP32-CAM
